### 설명
플레이어의 랭크를 설정하고 확인할 수 있는 플러그인입니다.

### 명령어
```/rank set <rank> [player]``` : [player]의 랭크를 설정합니다. (없으면 본인의 랭크를 설정합니다)

```/rank info [player]``` : [player]의 랭크를 확인합니다. (없으면 본인의 랭크를 확인합니다)
